# Weather App

A simple weather application built with vanilla JavaScript that fetches real-time weather data from the OpenWeatherMap API.

## Features

- **Search Weather by City** — Enter any city name to get current weather
- **Async/Await Implementation** — Uses modern JavaScript async functions to handle API calls
- **Error Handling** — Proper error messages for invalid cities, network errors, and API issues
- **Search History** — Saves searched cities in localStorage, click to search again
- **Event Loop Console** — Shows execution order of synchronous, asynchronous, and microtask code
- **Responsive Design** — Works on desktop and mobile devices

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- OpenWeatherMap
